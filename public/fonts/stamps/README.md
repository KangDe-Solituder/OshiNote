Stamp font assets
=================

Place optional stamp fonts here to enable font-dependent stamp materials.

Expected files:

- `great-vibes/GreatVibes-Regular.ttf`
  - Enables the English flourish stamp.
  - Source: https://raw.githubusercontent.com/google/fonts/main/ofl/greatvibes/GreatVibes-Regular.ttf
  - License: SIL Open Font License, see https://github.com/google/fonts/tree/main/ofl/greatvibes

- `yuji-syuku/YujiSyuku-Regular.ttf`
  - Enables the CJK calligraphy stamp.
  - Source: https://raw.githubusercontent.com/google/fonts/main/ofl/yujisyuku/YujiSyuku-Regular.ttf
  - License: SIL Open Font License, see https://github.com/google/fonts/tree/main/ofl/yujisyuku

After adding font files, use Settings -> Stamp Fonts -> Check Fonts.
